RRHH Chile - Odoo 12

Localización chilena de RRHH para Odoo 12

RRHH Chile - Solicitud de Término de Empleado

[l10n_cl_hr_finiquito_solicita]

Allows employees or managers to submit and approve termination requests.

Funciones

☑ Fecha de Antigüedad desde Contrato Empleado
☑ Motivos de Término según Código del Trabajo de Chile (Art. 159, 160 y 161)
☑ Workflow de Envío y Aprobación
☑ Determina los Días de Preaviso para Finiquito

Autores

Intellego-BI.com

Resumen

This addon will enable Odoo users to submit and approve employee termination requests. Once a request is submitted, a manager must approve it. Approved requests are required in order to calculate final settlement (finiquito) in Payroll.

Dependencias

  • Odoo Empleados [hr]
  • Odoo Mail [mail]
  • RRHH Chile Extensión Maestro de Empleados [l10n_cl_hr_employee_extend]
